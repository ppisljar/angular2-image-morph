Step 1: Upload two images you would like to morph
Step 2: Select matching points on the two images (in same order!)
Step 3: Set numbers of frames you want and click GENERATE
Congratulations. You can now save individual frames or a GIF animation

Whole website runs in your browser. Everything is completely client side. Images are never transfered to the server but are morphed with JavaScript.

Angular.JS 2.0 for framework - Gif.JS for GIF generation - Bootstrap CSS 3.3 for styling

You can find this whole website and image morphing library shared on GitHub
